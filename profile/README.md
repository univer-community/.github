# univer-community

`univer-community` is where the `Univer` ecosystem grows beyond the core.

We build community-maintained plugins, adapters, and tooling that help teams adopt `Univer` faster, integrate it into real-world products, and turn one-off experiments into reusable building blocks.

## What We Build

- Developer tooling that makes building standalone `Univer` plugins faster and more repeatable.
- Adapters that let existing applications bring `Univer` into mature stacks without a costly rewrite.
- Focused community projects that solve practical plugin and integration problems in the open.

## Featured Projects

- [`univer-community/cli`](https://github.com/univer-community/cli): An interactive scaffold for launching standalone `Univer` plugin projects with production-ready structure, configuration, and templates from day one.
- [`univer-community/ui-adapter-vue2`](https://github.com/univer-community/ui-adapter-vue2): A Vue 2 adapter that makes it possible to reuse existing Vue 2 components inside `Univer` UI surfaces such as sidebars and panels.

## Contributing

Ideas, issues, and pull requests are welcome.

If you are building a useful plugin, adapter, or workflow around `Univer`, this is the place to share it, improve it in public, and grow the ecosystem with the community.
